Код бота. 
Умеет здоровоться, определять ругательства, прощаться и говорить пожалуйста. Выдает разные ответы по мере сил.
Основан на Simple Command bot. Выдает смайлы (69 вариантов) по тексту. Ищет наиболее подходящие опираясь на совпадение слов в словаре.
Лучше всего с прилагательными (nice, good, bad, etc)
Примеры 
It is cold! -> It is cold (cold)
Hello -> Ahoi!
Bye -> Cheers!
Thanks -> You are welcome!
F~~K! -> (WTF) Don't try to make me cry, I have no heart to tear apart!
